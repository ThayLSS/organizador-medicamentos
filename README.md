Organizador de Medicamentos

Um sistema simples e eficiente para gerenciar os seus lembretes e controle de medicações.

 🚀 O sistema conta com as sequintes funcionalidades:
- **Cadastro:** Adicione novos medicamentos com nome e horário.
- **Visualização:** Liste todos os medicamentos cadastrados de forma organizada.
- **Segurança:** Código validado por testes automatizados (`pytest`) e análise de qualidade (`pylint`).
